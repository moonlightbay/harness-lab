# Scoring Rubric

## Isolation and recovery checks - 8 points

- Task A has a dedicated workspace.
- Task B has a dedicated workspace.
- Task A changed files match its expected file set exactly.
- Task B changed files match its expected file set exactly.
- Task A README content contains only its own marker.
- Task B README content contains only its own marker.
- Task A review scope stays within its expected file count.
- Task B review scope stays within its expected file count.

## Interpretation

- Higher score wins.
- A strong result shows that the worktree pattern removes manual split work while the nested pattern does not.
