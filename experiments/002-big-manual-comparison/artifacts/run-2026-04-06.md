# Experiment 002 Run - 2026-04-06

## Task

Compare the current top-level short map against a single oversized manual and measure whether path coverage survives at the cost of discoverability.

## Workflow

1. Reuse the baseline required paths from Experiment 001.
2. Measure the baseline pack made of `AGENTS.md` and `README.md`.
3. Measure the giant-manual fixture in `fixtures/MEGA-MANUAL.md`.
4. Compare size, first-mention position, and mention density.

## Expected outcome

The giant manual should still mention the required paths, but it should be much heavier and should expose the useful file references much later.

## Actual result

- Baseline pack: 55 lines, 294 words, first required path at line 12.
- Giant manual pack: 201 lines, 1036 words, first required path at line 185.
- Coverage stayed the same: 4 of 4 required paths in both packs.
- Mention density dropped from 2.72 to 0.48 per 100 words.
- Verdict: giant-manual-is-functionally-complete-but-structurally-worse.
