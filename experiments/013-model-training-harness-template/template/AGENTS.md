# AGENTS.md

## Repository purpose

This repository uses a lightweight scaffold for AI-assisted model training.

## Default read set

- `README.md`
- `docs/project.md`

Read `docs/log.md` for recent history. Open `configs/README.md`, `src/README.md`, and `scripts/README.md` only when changing those areas.

## Expectations

- Keep `AGENTS.md` short.
- Keep top-level guidance current and singular.
- Keep `docs/log.md` brief and append-only.
- Put configs in `configs/`, training code in `src/`, and runnable commands in `scripts/`.
- Keep long-lived project facts in docs, not only in chat.
