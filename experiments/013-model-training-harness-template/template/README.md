# Training Scaffold

A small docs-first starter for model training repos.

Source of truth for project context: `docs/project.md`
Source of truth for work log: `docs/log.md`

## Layout

- `configs/`: base, experiment, and debug configs
- `src/`: data, model, training, and evaluation code
- `scripts/`: reproducible entrypoints
- `docs/`: project context and short log
- `checks/`: lightweight guidance checks
