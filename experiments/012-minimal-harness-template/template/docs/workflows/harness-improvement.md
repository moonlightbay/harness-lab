# Harness Improvement

## Improvement loop

If the same friction happens repeatedly, do not keep solving it only in chat.

## Promotion targets

- promote it to a `rule` when humans keep repeating the same guidance
- promote it to a `check` when it can be enforced mechanically
- promote it to a `skill` when it is a high-frequency reusable operation

## Review habit

Review the harness after each major wave and add one small improvement if a repeated source of friction is clear.
